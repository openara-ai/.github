# openara.ai

**Agentic engineering, built in the open.**

openara.ai is an independent consulting and research practice focused on **agentic engineering**: using AI agents to shorten idea-to-ship cycles while maintaining production-quality engineering discipline.

The projects published here are the research arm of the practice. Each is a working tool, built end-to-end through agentic engineering workflows — using current commercial agent tools (Claude, Codex, Copilot, etc.) under human architectural direction. The projects, the build process, the observations, and the methodology are all open and publicly documented.

The Agentic Engineering research spans these areas:

- **Agents** — Agentic systems that act on a user's behalf, taking steps toward an outcome rather than just answering a question.
- **Reasoning** — Approaches that make the *why* behind a system's behavior visible, so users can see and redirect it.
- **Applications** — Practical end-user tools that solve real, everyday problems.
  
## Featured project

### [MediaSearchAgent](https://github.com/openara-ai/media-search-agent)

A local-first AI media search application for personal photos and videos. Explores semantic search, face search, video indexing, metadata extraction, and natural-language discovery across personal media collections.

**Built through agentic engineering.** The human drove requirements, architecture, reviews, and testing. AI coding tools assisted with research, implementation, refactoring, debugging, and documentation. The details are in the repository — what worked, what didn't, and where the human had to intervene.

## How we build

- **Useful before polished.** Ship the thing that solves the problem. Polish follows usefulness.
- **Transparent about limitations.** Say what works, what doesn't, and where the rough edges are. No magic, no hype.
- **Friendly to experimentation.** Designed to be poked at, forked, and rewired. Code as a starting point, not a black box.
- **Grounded in real workflows.** Start from how people actually use their files, photos, and tools.
- **Learning artifacts.** Each project is also a public record of how it was built, including the AI-assisted parts.
